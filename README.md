Hiểu cách viết tắt của css property:  
 property: a b c d  
 a: trên  
 b: phải
 c: dưới
 d: trái
 Nếu chỉ có a, b:  
  a: Trên dưới  
  b: Trái phải  
  Tương tự nếu có thêm c: c = trái phải
